# Code of Conduct

This project uses the Contributor Covenant Code of Conduct, version 2.1.

Participants are expected to be respectful, constructive, and focused on improving the project. Unacceptable behavior includes harassment, discriminatory language, personal attacks, and publishing private information without permission.

Report conduct concerns privately through the repository owner's GitHub contact or the repository security advisory path when the concern involves sensitive information.

Project maintainers may remove, edit, or reject contributions that do not align with this code of conduct.

For the full Contributor Covenant text, see:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/
